# Syntaxwear E-commerce

Este é um projeto de e-commerce de página única para uma loja de tênis chamada "Syntaxwear". O projeto foi construído com HTML e CSS e apresenta um design responsivo que se adapta a diferentes tamanhos de tela.

## Visão Geral

O site da Syntaxwear foi projetado para ser uma plataforma de e-commerce moderna e atraente para a venda de tênis. A página principal inclui várias seções:

-   **Cabeçalho**: Contém o logotipo da loja, um menu de navegação principal, um menu de acesso rápido e um ícone de menu hambúrguer para dispositivos móveis.
-   **Seção Hero**: Uma seção de destaque com uma imagem de fundo, um título chamativo e botões de chamada para ação (CTA).
-   **Seção de Categorias**: Apresenta as diferentes categorias de tênis disponíveis, como "Casual", "Esporte", "Moderno" e "Futurista".
-   **Grid de Produtos**: Uma grade de produtos em destaque que exibe os tênis mais populares da loja.
-   **Rodapé**: Inclui um formulário de inscrição para newsletter, links para as redes sociais da loja, um mapa do site e informações de direitos autorais.

## Estrutura do Projeto

O projeto está organizado da seguinte forma:

-   `index.html`: O arquivo principal que contém a estrutura da página.
-   `css/`: Uma pasta que contém todos os arquivos CSS do projeto.
    -   `base.css`: Contém os estilos base do corpo da página e dos botões.
    -   `layout.css`: Um arquivo vazio que provavelmente foi criado para conter os estilos de layout do site.
    -   `reset.css`: Um arquivo que reseta os estilos padrão do navegador para garantir uma aparência consistente em todos os navegadores.
    -   `variables.css`: Um arquivo que define as variáveis CSS do projeto, como a fonte principal.
    -   `components/`: Uma pasta que contém os arquivos CSS para cada componente da página, como o cabeçalho, a seção hero, as categorias de produtos, a grade de produtos e o rodapé.
-   `images/`: Uma pasta que contém todas as imagens do projeto, incluindo banners, ícones, logotipos e imagens de produtos.

## Como Usar

Para visualizar o site, basta abrir o arquivo `index.html` em seu navegador.

## Possíveis Melhorias

-   **JavaScript**: O site atualmente não usa JavaScript. A adição de JavaScript poderia melhorar a interatividade do site, por exemplo, adicionando um carrossel de produtos ou um formulário de contato funcional.
-   **Layout**: O arquivo `layout.css` está vazio. A criação de um layout mais robusto e flexível poderia melhorar a aparência e a usabilidade do site em diferentes dispositivos.
-   **Backend**: O site atualmente é estático. A adição de um backend poderia permitir a criação de um sistema de gerenciamento de conteúdo (CMS) para gerenciar os produtos e as categorias da loja.
